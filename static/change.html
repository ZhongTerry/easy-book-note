当前版本

v1.1.1 搜索重构与细节打磨

  • 聚合搜索重构： 引入 Owllook 聚合搜索接口作为核心引擎，配合 360 搜索多线程解密兜底，彻底解决云服务器 IP 被搜索引擎风控导致“搜书无结果”的顽疾。
  • CDN 穿透优化： 后端强制禁用 /read 路由的 HTML 缓存，彻底解决 EdgeOne 等 CDN 导致的“PC端误显示手机版页面”的缓存死锁问题。
  • 智能目录交互： 目录弹窗新增“自动定位”功能，打开即平滑滚动至当前章节；修复章节乱序问题（前端强制 ID 排序）；优化中文数字解析算法（修复“第十一章”识别错误）。
  • PC 视觉微调： 翻页按钮迭代为更克制的“胶囊”样式（130px），统一低饱和度配色以减少阅读干扰；深度修复黑暗模式下顶栏与模态框的色差问题。
  • 双重刷新机制： 在阅读器顶部和目录栏分别增加“强制刷新”入口，支持分别重载页面逻辑和重新抓取目录结构。

v1.1.0 极致极简体验升级 (Tomato Style)

  • PC端沉浸式重构： 深度参考“番茄小说”视觉规范，移除所有冗余导航栏。采用 900px 黄金阅读宽度，配合羊皮纸/丝绸质感背景，打造极致护眼的 PC 阅读环境。
  • 设备自适应分流： 后端引入智能 UA 识别逻辑，实现 reader_pc.html 与移动端模板彻底解耦，针对不同设备特性提供差异化的交互逻辑。
  • 毛玻璃目录抽屉： 为 PC 端新增半透明毛玻璃特效的侧边目录栏，支持实时定位当前章节，并整合“强制刷新目录”功能。
  • 功能高度整合： 独家 PureUI 交互升级，将字号调节、换源、TTS 朗读、背景切换等功能统一收纳至右侧悬浮设置面板，阅读区“零干扰”。
  • 全键盘操作支持： 新增对键盘左右方向键的监听，实现无需鼠标点击的极速翻页体验。
  • 细节体验打磨： 在页面底部新增微型信息栏（实时时钟与阅读进度），让阅读状态一目了然。

v1.0.5 跨越“墙”的阅读自由

  • PWA 全屏应用化： 接入 Web App Manifest 与 Service Worker，支持“添加到桌面”，彻底隐藏浏览器地址栏，沉浸感媲美原生 App。
  • 爬虫底层升级： 引入 curl_cffi 模拟真实浏览器 TLS 指纹，配合系统级代理调度，无视 Cloudflare 等高级防火墙阻碍。
  • 自动缝合技术： 针对分页小说章节实现后端自动递归抓取并无缝拼接，用户无需点击“下一页”即可通读全章。

v1.0.4 追更系统的“量化感知”

  • 数字化追更： 实时计算“最新 ID - 当前 ID”，前端直接显示 +14 等未读章节数，点击 🔄 即可一键触达。
  • 智能书名补全： 即使手动输入 URL，系统也会通过面包屑导航算法自动拾取中文书名，告别拼音 Key 的困扰。
  • 多引擎搜索增强： 实现了 DuckDuckGo 与 Bing 国际版双引擎切换，并支持系统自动代理设置，搜书效率提升 200%。

v1.0.3 数据主权与书单看板

  • 版本回退功能： 针对手动填写的 Value 记录最近 5 次修改历史，提供“后悔药”功能，支持一键还原历史节点。
  • 阅读历史记录： 自动记录最近 50 条阅读足迹，支持按书名 Key 自动置顶与去重，方便随时“断点续读”。
  • 书单看板系统： 支持新建多个自定义书单，采用两栏式看板布局（想看/已看），并支持在书单中一键调取目录。

v1.0.2 工业级架构重构

  • 数据大一统： 编写 migrate.py 脚本，将分散的 JSON 和用户 SQLite 迁移至中心化的 data.sqlite，大幅降低 IO 损耗。
  • 管理仪表盘： 为管理员增加专属面板，实时监控服务器 CPU、内存、磁盘占用及缓存文件规模。
  • 高级权限体系： 实现 Admin / Pro / User 三级鉴权，为 Pro 用户提供全本离线下载（Json Package）特权。
